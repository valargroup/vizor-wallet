мҡ°лҰ¬ м ‘к·јлІ•мқҖ librustzcashлҘј FFIлЎң м§Ғм ‘ нҳём¶ңн•ҳл©°, мқҙлҘј нҶөн•ҙ:

  • flutter_rust_bridgeк°Җ нҒ¬лЎңмҠӨ м»ҙнҢҢмқј, async, нғҖмһ… л§Ҳмғ¬л§Ғмқ„ мһҗлҸҷмңјлЎң мІҳлҰ¬

2. Flutter Rust Bridge (FRB) нҶөмӢ 

2.1 FRB мһ‘лҸҷ л°©мӢқ

FRBлҠ” rust/src/api/мқҳ Rust мҶҢмҠӨ нҢҢмқјмқ„ мқҪкі  Dart л°”мқёл”©мқ„ мғқм„ұн•ңлӢӨ. лҹ°нғҖмһ„м—җ:

  1. Dartк°Җ мғқм„ұлҗң async н•ЁмҲҳлҘј нҳём¶ң (мҳҲ: createWallet())
  2. FRBк°Җ SSE(Simple Serialization Engine)лЎң мқёмһҗлҘј м§Ғл ¬нҷ”
  3. лі„лҸ„ isolate/threadмқҳ Rust н•ЁмҲҳлЎң нҳём¶ңмқҙ м „лӢ¬лҗЁ
  4. Rustк°Җ н•ЁмҲҳлҘј мӢӨн–үн•ҳкі  кІ°кіјлҘј м§Ғл ¬нҷ”
  5. FRBк°Җ кІ°кіјлҘј м—ӯм§Ғл ¬нҷ”н•ҳм—¬ Dart к°қмІҙлЎң л°ҳнҷҳ

#[flutter_rust_bridge::frb(sync)]лЎң н‘ңмӢңлҗң Rust н•ЁмҲҳлҠ” нҳём¶ң мҠӨл Ҳл“ңм—җм„ң лҸҷкё°м ҒмңјлЎң мӢӨн–үлҗңлӢӨ(isolate м „нҷҳ м—ҶмқҢ). к°„лӢЁн•ҳкі  лё”лЎңнӮ№н•ҳм§Җ м•ҠлҠ” м—°мӮ°м—җл§Ң мӮ¬мҡ©н•ҙм•ј н•ңлӢӨ.

2.4 м„Өкі„ м ңм•Ҫ

  • API кІҪкі„м—җлҠ” лӢЁмҲң нғҖмһ…л§Ң мӮ¬мҡ©. FRBлҠ” н”„лҰ¬лҜёнӢ°лёҢ, String, Vec, н”Ңлһ« structлҘј мһҳ мІҳлҰ¬н•ңлӢӨ. ліөмһЎн•ң Zcash нғҖмһ…(WalletDb, UnifiedSpendingKey, CompactBlock)мқҖ Rust лӮҙл¶Җм—җ лЁёл¬јлҹ¬м•ј н•ңлӢӨ. API л Ҳмқҙм–ҙ(rust/src/api/)к°Җ лӢЁмҲң нғҖмһ…кіј лӮҙл¶Җ Zcash нғҖмһ… к°„мқҳ ліҖнҷҳмқ„ лӢҙлӢ№н•ңлӢӨ.
  • нҳём¶ң к°„ Rust мғҒнғңлҘј мң м§Җн•ҳм§Җ м•ҠлҠ”лӢӨ. к°Ғ FFI нҳём¶ңмқҖ DBлҘј м—ҙкі , м—°мӮ°мқ„ мҲҳн–үн•ҳкі , лӢ«лҠ”лӢӨ. Rustм—җ мһҘкё° мң м§ҖлҗҳлҠ” WalletDb мқёмҠӨн„ҙмҠӨк°Җ м—ҶлӢӨ. лҚ” лӢЁмҲңн•ҳкі  лқјмқҙн”„мӮ¬мқҙнҒҙ л¬ём ңлҘј н”јн•  мҲҳ мһҲлӢӨ.
  • м—җлҹ¬лҠ” л¬ёмһҗм—ҙмқҙлӢӨ. API м „мІҙм—җм„ң Result<T, String>мқ„ мӮ¬мҡ©н•ңлӢӨ. Zcash crate м—җлҹ¬лҠ” API кІҪкі„м—җм„ң мӮ¬лһҢмқҙ мқҪмқ„ мҲҳ мһҲлҠ” л¬ёмһҗм—ҙлЎң нҸ¬л§·лҗңлӢӨ.

3.2 Wallet API вҖ” нӮӨ мң лҸ„ & кі„м • кҙҖлҰ¬

нҢҢмқј: rust/src/wallet/keys.rs

лӢҲлӘЁлӢү мғқм„ұ: bip0039 нҒ¬л ҲмқҙнҠёлЎң 256л№„нҠё м—”нҠёлЎңн”јмқҳ 24лӢЁм–ҙ BIP-39 лӢҲлӘЁлӢүмқ„ мғқм„ұн•ңлӢӨ. Seed мң лҸ„лҠ” н‘ңмӨҖмқ„ л”°лҘёлӢӨ: PBKDF2-HMAC-SHA512, salt = "mnemonic" + passphrase (л№Ҳ passphrase).

нӮӨ мң лҸ„: zcash_keys::UnifiedSpendingKey::from_seed()к°Җ н•ҳлӮҳмқҳ seedм—җм„ң м„ё мў…лҘҳмқҳ нӮӨлҘј мң лҸ„н•ңлӢӨ:

  • Transparent: BIP-44 кІҪлЎң m/44'/133'/0' (mainnet)
  • Sapling: Jubjub кіЎм„  мң„мқҳ ZIP-32 HD мң лҸ„
  • Orchard: Pallas кіЎм„  мң„мқҳ ZIP-32 HD мң лҸ„

кі„м • мғқм„ұ: WalletDb::create_account()мқҖ Unified Full Viewing Key (UFVK)л§Ң лҚ°мқҙн„°лІ мқҙмҠӨм—җ м ҖмһҘн•ңлӢӨ. Spending keyлҠ” DBм—җ мҳҒмҶҚнҷ”н•ҳм§Җ м•ҠлҠ”лӢӨ.

мЈјмҶҢ мқёмҪ”л”©: Unified AddressлҠ” Require Orchard + Require Sapling + Omit TransparentлЎң мғқм„ұн•ҳл©°, zodl/Zashi мӣ”л ӣ лҸҷмһ‘кіј мқјм№ҳн•ңлӢӨ. shielded_address_request() н—¬нҚј н•ЁмҲҳк°Җ мқҙлҘј м ңм–ҙн•ңлӢӨ.

fn shielded_address_request() -> UnifiedAddressRequest {
    UnifiedAddressRequest::custom(
        ReceiverRequirement::Require, // Orchard
        ReceiverRequirement::Require, // Sapling
        ReceiverRequirement::Omit,    // Transparent
    ).expect("valid receiver requirements")
}

3.3 Sync API вҖ” лё”лЎқ мҠӨмәҗлӢқ & мһ”м•Ў

нҢҢмқј: rust/src/wallet/sync.rs

Dartм—җ л…ём¶ңлҗҳлҠ” к°ңлі„ sync лӢЁкі„. к°Ғ н•ЁмҲҳлҠ” DBлҘј м—ҙкі , н•ҳлӮҳмқҳ м—°мӮ°мқ„ мҲҳн–үн•ҳкі , л°ҳнҷҳн•ңлӢӨ. Dartк°Җ мӢңнҖҖмҠӨлҘј мҳӨмјҖмҠӨнҠёл Ҳмқҙм…ҳн•ңлӢӨ.

Functionм—ӯн• 
update_chain_tip()мӣ”л ӣ DBм—җ мөңмӢ  лё”лЎқ лҶ’мқҙлҘј м„Өм •н•ңлӢӨ. suggest_scan_ranges() м „м—җ л°ҳл“ңмӢң нҳём¶ңн•ҙм•ј н•ңлӢӨ.
put_sapling_subtree_roots()нҡЁмңЁм Ғ нҠёлҰ¬ мһ¬кө¬м„ұмқ„ мң„н•ң Sapling commitment tree subtree rootлҘј м ҖмһҘн•ңлӢӨ.
put_orchard_subtree_roots()Orchardм—җ лҢҖн•ҙ лҸҷмқј.
suggest_scan_ranges()мӣ”л ӣ DBм—җм„ң мҠӨмә”мқҙ н•„мҡ”н•ң лё”лЎқ лІ”мң„лҘј мҡ°м„ мҲңмң„ мҲңмңјлЎң мЎ°нҡҢн•ңлӢӨ.
write_block_metadata()FsBlockDb л©”нғҖлҚ°мқҙн„° DBм—җ compact block нҢҢмқјмқ„ л“ұлЎқн•ңлӢӨ. нҢҢмқј мЎҙмһ¬ м—¬л¶ҖлҘј кІҖмҰқн•ңлӢӨ.
scan_blocks()zcash_client_backendмқҳ scan_cached_blocks()лҘј мӮ¬мҡ©н•ҳм—¬ мәҗмӢңлҗң compact blockм—җ trial decryptionмқ„ мҲҳн–үн•ңлӢӨ.
get_sync_progress()мӣ”л ӣ мҡ”м•Ҫм—җм„ң fully_scanned_heightмҷҖ chain_tip_heightлҘј л°ҳнҷҳн•ңлӢӨ.
get_wallet_balance()transparent, Sapling, Orchard н’Җмқҳ мӮ¬мҡ© к°ҖлҠҘ мһ”м•Ўмқ„ л°ҳнҷҳн•ңлӢӨ.
