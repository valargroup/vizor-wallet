# Vizor Wallet v0.0.17

This release improves the send review experience, duplicate account import
feedback, and macOS update checks.

## Improvements

- Refined the send review screen in dark mode with updated receipt artwork and
  better color treatment.
- Made the final send confirmation easier to scan with clearer transaction fee
  labeling and button layout.
- Improved duplicate import messages for Secret Passphrase and Keystone
  accounts so Vizor now clearly explains when an account is already in your
  wallet.
- macOS automatic update checks now run more frequently.

## Fixes

- Fixed a macOS issue where the Check for Updates menu item could stay inactive
  when app updates were correctly configured.
