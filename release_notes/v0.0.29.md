# Vizor Wallet v0.0.29

This release keeps Vizor compatible with the latest Zcash network upgrade and
improves wallet recovery when shielded sync data needs to be repaired.

## What's New

- Transaction details, send confirmations, and swap receipts can now show saved
  address book labels when a matching contact is available.

## Improvements

- Added support for the latest Zcash network upgrade so Vizor can continue
  syncing and sending on the upgraded chain.
- Improved wallet startup reliability by ensuring required wallet database
  migrations run before the app reads account, balance, or activity data.
- Refreshed the ZEC logo used in swap screens.

## Fixes

- Improved recovery for wallets whose shielded sync state becomes inconsistent,
  including cases that could prevent ZEC sends from reaching the network.
- Fixed additional sync completion edge cases so Vizor does not report a wallet
  as fully synced while repair work is still needed.
- Improved failed activity row styling so failed items blend correctly with the
  activity list.

## Notes

- After updating, Vizor may rescan recent Zcash blocks if it detects shielded
  wallet state that needs repair. Let sync finish before retrying a failed send.
