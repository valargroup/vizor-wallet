# Vizor Wallet v0.0.19

This release improves Keystone onboarding and QR scanning, adds clearer sync
visibility, strengthens account removal safety, and hardens local Secret
Passphrase handling on macOS.

## What's New

- Added a sync status indicator to the sidebar so you can see whether Vizor is
  syncing, synced, or needs attention while moving around the app.
- Added password confirmation before removing an account or resetting Vizor
  from the account removal flow.
- The About Vizor screen now includes clearer product information plus links to
  the Vizor website and GitHub.

## Improvements

- Keystone onboarding now gives clearer Cypherpunk firmware and connection
  steps, including the Zodl-compatible connection path.
- Keystone QR scanning now includes a Trouble scanning guide, clearer camera
  permission recovery, better camera selection, and more specific scan
  instructions.
- Secret Passphrase and Keystone imports now warn before scanning from the
  earliest supported shielded height when you cannot remember the wallet
  birthday.
- Recent Activity on Home now shows up to 10 transactions and keeps sync status
  separate from the activity feed.
- Activity amounts now use more compact precision, avoiding unnecessary trailing
  decimals while preserving small-value detail.
- Account management now has clearer Current and Other sections, a refreshed Add
  Account action, and visible Keystone labels for hardware accounts.
- macOS software sends and shielding now use the locally stored Secret
  Passphrase through the native Rust path, reducing sensitive data exposure in
  the UI layer.

## Fixes

- Keystone receive screens no longer show the shielded-address Renew action for
  hardware accounts, since Keystone accounts use a fixed shielded address.
- Light destructive buttons now use the correct label color.
