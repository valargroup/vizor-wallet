# Vizor Wallet v0.0.18

This release improves account switching, wallet import timing, Keystone QR
scanning, transaction receipts, and macOS Secret Passphrase storage reliability.

## Improvements

- Switching accounts now returns you to the wallet home screen immediately while
  balances and activity refresh in the background.
- The wallet birthday calendar opens without waiting for endpoint metadata, so
  Secret Passphrase and Keystone imports feel more responsive.
- Keystone QR scanning on macOS is more reliable, with QR-only detection, a
  larger scan target, and a focused scan window.
- Send review and transaction receipt screens now scale long amounts to fit
  cleanly and use clearer transaction fee labeling.

## Fixes

- Improved macOS Secret Passphrase storage with automatic migration to a
  dedicated secure storage area after unlock.
- Wallet reset and password cleanup now wait for in-progress secret writes,
  reducing the chance of stale secure-storage data after a reset.
- Transaction detail receipts now show amount formatting more consistently with
  the rest of the activity UI.
