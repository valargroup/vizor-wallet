# Vizor Wallet v0.0.20

This release improves clarity around sending your full spendable balance and
polishes sync status visibility in dark mode.

## What's New

- Added a help tooltip next to Max on the Send screen explaining why spendable
  balance can be lower than total balance.

## Improvements

- Refined sidebar sync status indicator colors in dark mode so synced and
  attention states are easier to read.
