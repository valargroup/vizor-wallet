# Vizor Wallet v0.0.22

This is a small update-test release for Vizor. It exists to make sure the app
can move cleanly from one version to the next before larger changes ship.

## What's New

- Added a new version number so Vizor has something fresh to update to. Small
  job, important job.

## Notes

- No wallet features, balances, accounts, or transaction behavior are intended
  to change in this release.
- After updating, Vizor should continue from the same wallet state as before.
