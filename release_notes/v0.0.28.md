# Vizor Wallet v0.0.28

This release adds mainnet swaps and an address book, with better address entry
for cross-chain payments.

## What's New

- Added Swap on mainnet, powered by NEAR Intents, for exchanging ZEC with
  supported external assets. The Swap flow includes live quotes, fiat amount
  entry, slippage settings, activity tracking, status pages, refund and timeout
  states, and QR-based Keystone signing for ZEC deposits.
- Added an address book for saving, searching, editing, and reusing contacts
  across Zcash and supported external networks, with QR scanning, contact
  picking, and per-network address checks for common Zcash, EVM, Bitcoin,
  Solana, and NEAR address formats.

## Fixes

- Fixed transparent transaction fee backfill so wallet activity can show fees
  more accurately when spend evidence is available.
