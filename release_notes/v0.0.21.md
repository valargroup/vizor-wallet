# Vizor Wallet v0.0.21

This release cleans up wallet creation state after setup and polishes spendable
balance guidance.

## Improvements

- The spendable balance help tooltip on the Send screen is easier to read in
  dark mode, with a clearer highlighted title.
- Wallet creation now clears temporary recovery phrase state after setup
  completes, so returning to onboarding starts from a clean state.
