# Vizor Wallet v0.0.26

This release improves Keystone receive addresses, transaction explorer links,
and shielding recovery behavior.

## Improvements

- Transaction explorer links now open CipherScan for mainnet and testnet
  transactions.
- Keystone accounts can now generate a fresh shielded receive address from the
  Receive screen.

## Fixes

- Fixed retryable shielding broadcast failures so locally created shielding
  transactions are queued for retry and surfaced in Activity instead of only
  showing a generic failure.
